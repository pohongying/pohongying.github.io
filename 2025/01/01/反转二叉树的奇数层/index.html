    2415 .反转二叉树的奇数层

    方法一：dfs

    思路

    这道题很有意思，之前没说反转的奇数层，因此可以后序遍历，可以实现交换两个节点。而这道题要注意的点事，它只能前序遍历，因此它只能交换节点的值，并且要实现奇偶装换，这里很巧妙，它用了！ 来实现，还有就是它比较的是左右两个子树，而不是同一个子树

    代码实现

    class Solution {
    boolean flag = true;

    public TreeNode reverseOddLevels(TreeNode root) {
        travel(root.left,root.right, flag);
        return root;

    }

    public void travel(TreeNode left,TreeNode right, boolean flag) {
        if (left == null) {
            return ;
        }
        if (flag) {
            int val = left.val;
            left.val = right.val;
            right.val = val;
        }
        travel(left.left,right.right,!flag);
        travel(left.right,right.left, !flag);

    }

}

    方法二：bfs

    思路

    同样是，奇偶互换，设置一个flag

    代码实现

    class Solution {
    public TreeNode reverseOddLevels(TreeNode root) {
        Deque<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        boolean flag = false; // 作为标识很重要
        if (root == null) {
            return root;
        }

        while (!queue.isEmpty()) {
            int size = queue.size();
            List<TreeNode> arr = new ArrayList<TreeNode>();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                if (flag) {
                    arr.add(node);
                }
                if (node.left != null) {
                    queue.offer(node.left);
                    queue.offer(node.right);

                }

            }
            if (flag) {
                for (int l = 0, r = size - 1; l < r; l++, r--) {
                    int temp = arr.get(l).val;
                    arr.get(l).val = arr.get(r).val;
                    arr.get(r).val = temp;
                }
            }

            flag = !flag; // 每到下一层，进行反转

        }
        return root;

    }
}
